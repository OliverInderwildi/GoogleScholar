# Auto-updating Google Scholar HTML page

This package gives you a static HTML page that is regenerated by GitHub Actions.

## What it does
- pulls profile data from Google Scholar using the author ID `zy1UxxUAAAAJ`
- rebuilds `index.html`
- also writes `scholar-data.json`
- runs weekly and monthly, plus manually on demand

## Important limitation
A plain HTML file cannot reliably fetch Google Scholar directly in the browser because Google Scholar does not provide a public API for this use case and often blocks automated access. So the reliable pattern is:

1. GitHub Action runs on a schedule
2. Python fetches the Scholar profile
3. HTML is regenerated
4. GitHub Pages serves the updated static page

## Files
- `template.html` — page template
- `refresh_scholar.py` — fetch + render script
- `.github/workflows/refresh-scholar.yml` — scheduler
- `scholar-data.json` — generated profile data
- `index.html` — generated homepage file
